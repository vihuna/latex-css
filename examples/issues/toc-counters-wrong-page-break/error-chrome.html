LaTeX.css Paged Layout

Vincent Dörig
May 2020

Abstract

LaTeX.css is an almost class-less CSS library that turns your HTML document into a website that looks like a LaTeX document. Write semantic HTML, add <link rel="stylesheet" href="https://latex.now.sh/style.css"> to the <head> of your project and you are good to go. The source code can be found on GitHub at https://github.com/vincentdoerig/latex-css.

This is a paged CSS layout for LaTeX.css, using Paged.js. Both together display the document paginated in the screen, and allow to export it to PDF format through the browser.
